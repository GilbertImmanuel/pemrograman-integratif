# Implementasi CRUD GRPC dan protobuf dengan NODE JS
## Package yang digunakan:
* Node.js
* XAMPP/mySQL

## Cara pemakaian:
  - Masuk ke direktori
  - Menjalankan ```npm install```
  - Menjalankan ```node server.js```
  - Melakukan testing dengan aplikasi seperti Postman, dll.
